# 原型需求确认问题清单

生成原型前，按此清单逐项确认需求细节，避免返工。

## 第一轮：基础信息确认（3-4个问题）

| 确认项 | 问题示例 |
|--------|---------|
| 页面类型 | 这是列表页、详情页、表单页还是看板页？ |
| 主题风格 | 使用默认主题还是南航(CSA)主题？ |
| 布局方式 | 侧边栏布局还是顶部导航布局？ |
| 目标用户 | 这个页面主要给什么角色使用？ |

## 第二轮：功能细节确认（根据页面类型，3-5个问题）

### 列表页类
- 需要哪些筛选条件？是否需要高级筛选？
- 表格需要哪些列？哪些列需要固定？
- 是否需要批量操作？具体有哪些操作？
- 是否需要排序？默认按什么排序？
- 分页方式是前端分页还是后端分页？

### 详情页类
- 需要展示哪些信息分组？
- 是否需要操作按钮？有哪些操作？
- 是否需要关联数据展示？
- 是否需要时间线/操作记录？

### 表单页类
- 有哪些必填字段？
- 是否有字段联动关系？
- 是否需要分步表单？
- 提交后跳转到哪里？

### 看板页类
- 需要哪些统计指标？
- 是否需要图表？什么类型的图表？
- 是否需要快捷操作入口？

### 甘特图/排程类
- 时间维度是什么（小时/天/周）？
- 有哪些类型的条（停场条、航班条等）？
- 条上需要展示什么信息？
- 是否需要拖拽功能？
- 是否有特殊的视觉效果（闪烁、斑马纹等）？

## 第三轮：交互/数据细节确认（如需要，2-3个问题）

### 交互细节
- 点击某项后是跳转还是弹窗？弹窗内容是什么？
- 数据有哪些状态？状态之间如何流转？
- 是否需要动画效果？如淡入、滑动、闪烁等
- 操作成功后如何提示？Toast还是弹窗？
- 无数据时显示什么？

### 数据结构
- 请列出所有需要展示的字段名称
- 能否提供几条示例数据？
- 日期、金额等字段是什么格式？
- 状态字段有哪些可选值？各对应什么颜色？

### 视觉细节
- 是否有需要特殊强调的颜色？如警告用红色
- 是否需要图标？使用什么图标库？
- 是否有特殊的间距或布局要求？
- 是否需要考虑响应式适配？

## 确认完成标志

以下信息全部明确后，方可开始生成：

- [ ] 页面类型和用途明确
- [ ] 主题和布局确定
- [ ] 所有字段/列明确
- [ ] 主要交互方式确定
- [ ] 特殊效果/动画要求明确
- [ ] 用户确认需求理解正确
