## 关于本仓库

<p style="text-indent: 2em;">这个仓库是我在个人开发实践中积累的技能集合。这些技能是我在日常开发工作中逐步打磨出来的，涵盖了 Docker 构建、前端开发、Go 开发、代码审查等多个领域。它们帮助我提高了开发效率，规范了工作流程，现在我将这些技能开放出来，希望能对其他开发者也有所帮助。</p>

4. **自定义调整**：根据你的实际需求修改和调整

## 创建新技能

创建一个技能很简单——只需要一个包含 `SKILL.md` 文件的文件夹。你可以参考本仓库中的任何技能作为起点：

```markdown
---
name: my-skill-name
description: 清晰描述这个技能的作用和使用场景
---

# 我的技能名称

[在这里添加技能的具体说明和使用指南]

## 使用示例
- 示例用法 1
- 示例用法 2

## 使用规范
- 规范 1
- 规范 2
```

前置信息只需要两个字段：
- `name` - 技能的唯一标识符（小写，用连字符代替空格）
- `description` - 技能的完整描述，说明它做什么以及何时使用
